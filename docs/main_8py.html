Detailed Description

This main file awaits a proportional gain to arrive over serial, and then drives a 12V Pololu 37Dx70L 50:1 Gear motor connected to a nerf turret term project 180 degrees using a closed loop proportional controller class.

The response is recorded and sent back over Serial to be plotted on a PC side GUI.
